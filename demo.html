AppPay React Demo

This demo shows how to use the @apppay/react package.

Usage:
npm install @apppay/react thirdweb

Code:
import { PaymentModal } from '@apppay/react';

<PaymentModal
  paymentId="your-payment-id"
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  onPaymentComplete={handleSuccess}
/>

Note: This demo requires a real payment ID from your AppPay backend and a Web3 wallet like MetaMask.
